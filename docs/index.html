NextMeet

Today's meeting links, right in the menu bar.

A tiny macOS app that reads Calendar, finds join links, and opens the right meeting app when you click.

Download for macOS View on GitHub

Requires macOS 14 or newer.

  1. Unzip the download and move NextMeet to Applications.
  2. Open it and grant Calendar access.
  3. Use the calendar icon in the menu bar.
  4. If macOS warns, Control-click and choose Open.
